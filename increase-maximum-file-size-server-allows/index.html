Increase Maximum File Size Server Allows

  • ETdoFresh

ETdoFresh

5 July 2020 • 1 min read

When setting up a blog server, as we did in Install Ghost on Linode Docker, you may have realized that sometimes the server does not like huge files. In order to up the maximum file size allowed, edit your ~/nginx-conf/default.conf

server {
    ...
    client_max_body_size 500M;
}

Then restart your, in this site’s case

docker exec -it $DOCKER_POD_ID service nginx restart
